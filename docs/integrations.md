# Integrations

All the ways to integrate Claude Code into your workflow: the **Agent SDK** for programmatic access from TypeScript and Python, **GitHub Actions** for CI/CD pipelines, **hooks** as an integration mechanism, and cloud provider support via Bedrock, Vertex AI, and Azure AI Foundry.
